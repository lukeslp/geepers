## Python Data Pipelines

### Strange Places
**File**: `interactive/strange-places/scripts/build_data_v5.py`
**Output**: `data/phenomena.json` (113.7 MB, 113,763 records)
**Sources**:
- CORGIS UFO dataset (pre-geocoded)
- BFRO Bigfoot database (pre-geocoded)
- USGS earthquake catalog (2000-2025)
- NASA meteorite data (TidyTuesday mirror)
- NTSB aviation accidents
- Shadowlands haunted places
- USGS volcano data
- EPA Superfund sites

### Inequality Atlas
**File**: `interactive/inequality-atlas/scripts/run_pipeline.sh`
**Pipeline**:
1. `01_fetch_sources.py` → Fetches from Census API
2. `02_merge_datasets.py` → Merges by FIPS
3. `03_normalize_values.py` → Creates z-scores
4. `04_create_indexes.py` → Calculates ICI

**Output**:
- inequality_atlas_final.csv (3,143 counties)
- inequality_atlas_final_metadata.json
