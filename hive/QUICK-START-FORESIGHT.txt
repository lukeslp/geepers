================================================================================
KEY ENDPOINTS
================================================================================

GET  /api/health/providers
     - Test which providers are initialized
     - Returns: {healthy: bool, providers: {role: {status, provider, type}}}
     - Time: <1 second

POST /api/cycle/start
     - Trigger immediate prediction cycle
     - Returns: {status: "cycle_triggered", ...}
     - Time: Starts immediately in background

GET  /api/current
     - Get latest predictions
     - Returns: {cycle: {...}, predictions: [...]}
     - Time: <100ms

GET  /api/worker/status
     - Check worker health
     - Returns: {running: bool, is_healthy: bool, seconds_since_last_cycle: int}
     - Time: <100ms

GET  /api/stream
     - Server-Sent Events stream
     - Updates: predictions, cycles, heartbeats
     - Time: Real-time connection

================================================================================
TROUBLESHOOTING
================================================================================

Providers not working:
  1. Check /api/health/providers endpoint
  2. Look for error messages in app logs
  3. Verify API keys are set: XAI_API_KEY, ANTHROPIC_API_KEY, GEMINI_API_KEY
  4. Check requirements are installed: pip install anthropic google-generativeai openai

No predictions after 30s:
  1. Check /api/health/providers (may have failed provider)
  2. Check app logs for discovery/prediction errors
  3. Check /api/worker/status (is_healthy: true?)
  4. Try manual cycle: curl -X POST http://localhost:5062/api/cycle/start

Stock discovery fails:
  1. Check that xai provider is configured
  2. Check XAI_API_KEY environment variable
  3. Look for "Error discovering stocks" in logs

Empty grid stays empty:
  1. Wait 30 seconds from app startup
  2. Check browser console for JavaScript errors
  3. Check SSE connection: curl http://localhost:5062/api/stream
