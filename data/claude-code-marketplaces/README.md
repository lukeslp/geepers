# Claude Code Marketplaces Research Data

This directory contains structured data from research into Claude Code plugin marketplaces and submission requirements.

## Files

### marketplaces.json
Structured data about available marketplaces, including:
- Official Anthropic marketplace
- Community marketplaces (jeremylongshore, ananddtyagi, puerto)
- Awesome lists with star counts and priorities
- Infrastructure repositories worth monitoring

### submission-checklist.md
Complete checklist for submitting plugins to various marketplaces, derived from:
- Analysis of successful PRs to anthropics/claude-plugins-official
- Community marketplace CONTRIBUTING.md files
- PR templates and patterns from 10+ open submissions

### awesome-lists.json (to be created)
Detailed information about awesome lists for visibility.

## Research Summary

**Total marketplaces identified:** 4 primary + multiple awesome lists
**Anthropic official plugins:** 28 (13 internal, 15 external)
**Open PRs to official:** 10 (as of 2025-12-18)
**Community plugins available:** 243+ (jeremylongshore marketplace)

## Key Findings

1. **No formal CONTRIBUTING.md at Anthropic** - Process derived from PR analysis
2. **Multiple submission paths** - Official takes weeks/months, community faster
3. **Quality bar exists** - Security and documentation requirements clear
4. **Awesome lists provide visibility** - hesreallyhim/awesome-claude-code has 18k+ stars
